Prediction of average speed of a road by timestamp
------------------------------------------------------------------
Programming languages: Python 3

Required packages:
- pandas
- numpy
- date
- xgboost
- matplotlib
- sklearn
------------------------------------------------------------------
How to run:

1. Download the input files:
- Train.csv
- Test_input.csv
- Rainfall.csv
- PublicHoliday.csv
- Typhoon.csv
- Traffic congestion.csv
- Traffic_ClearWaterBay.csv
- Peak.csv

2. Just run through the whole pgm. To save time, you make wish to skip the 
   process of tuning parameters. You can stop after spliting the data into 
   training and testing datasets, and further continue the session 
   'Selected Best Model'.
   
3. Output file: test.csv
